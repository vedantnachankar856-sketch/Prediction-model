# 🚀 ML Prediction Model App

This is a Machine Learning web application built using **Streamlit** that predicts outcomes based on user input.

🔗 **Live App:** https://vedant-prediction-model.streamlit.app/  
🔗 **LinkedIn:** https://www.linkedin.com/in/vedant-nachankar-6396783b1  

---

## 📌 Project Overview

This project demonstrates how a trained Machine Learning model (`model.pkl`) can be deployed as an interactive web application.

Users can input values and get instant predictions using the trained model.

---

## ⚙️ Tech Stack

- Python 🐍  
- Streamlit 📊  
- Scikit-learn 🤖  
- NumPy & Pandas
